# JavaScriptCallCpp


JavaScript 调用 C++ 生成的 DLL，DLL 使用 ATL 封装的 ActiveX 控件，仅支持 IE 浏览器。

本资源为整套工程，编译 ATL_API 后，在Debug/Release目录下，使用 IE 打开TestJavaScript.html，点击“Add”按钮查看结果。

文档位置：https://download.csdn.net/download/bqrmt/10698732
